# 业余弄了个小玩具：帮你审合同的 Word/WPS 插件 v2.0

---

平时帮朋友看合同，或者自己签些杂七杂八的文件时，总觉得一字一句抠条款挺费眼睛的。

之前业余时间写了个代码小脚本，能把 AI 塞进 Word 里帮忙挑刺。没想到丢到网上后，真有不少小伙伴在用。于是最近又抽空把它重构了一下，算是正式的 **v2.0 个人版** 吧。

大家随便用，就当是个写着玩的小工具。

---

## 🛠 它能干点啥？

核心功能没变，还是**直接嵌在你的 Word 或 WPS 侧边栏里**当个随身小助手。

### 1. 帮你找找合同里的坑

不用在网页和 Word 之间来回复制了。点一下右边的「开始审查」，它会大概扫一眼：

- 权利义务是不是只向着一边
- 有没有漏掉什么关键条款（比如违约责任、管辖地）
- 数字或条款逻辑紧不紧凑

它会把觉得有问题的地方在原段落高亮出来，然后给个大概的修改建议。当个初筛工具用还是挺省事的。

### 2. 划选问答，省得去百度了

遇到那些又长又绕口的法律名词，直接在文档里用鼠标把那段话**划选**出来，然后在侧边栏直接问它：“这段话对我有什么坏处？”。
平时用它改改生硬的措辞也挺好用。

### 3. 可以教它“屁股决定脑袋”

写了个小开关，你可以告诉它现在是站在甲方还是乙方立场。它挑骨头的时候就会顺着你的立场来找茬。

## 🚀 这次 v2.0 填了哪些自己刨的坑？

### 1. Mac 和 Windows 终于都凑合能跑了

之前写得比较糙，很多用 Mac 的朋友反映跑不起来。
这次稍微花了点时间，把启动脚本重写了。现在不论是 **Windows 还是 Mac**，不管是装的 **Word 还是 WPS**，直接下载一份代码，双击运行，它基本都能自己顺藤摸瓜配好环境。填补了之前一堆“权限不足”、“命令找不到”的初级 Bug。

### 2. 可以捏自己的小模板

每个人的看合同习惯不一样。所以我加了个自定义指令的小功能，你可以把自己平时审租房合同、离职协议的心得写成一段 Prompt，单独存成一个专属模板。下次选这个模板，它就按你的习惯来挑刺。

### 3. 能本地离线跑了（防数据泄露）

如果你签的东西比较敏感，不放心传给网上的 AI，可以接个叫 **Ollama** 的东西。这就等于把 AI 安装在你自己电脑里了，拔掉网线也能在这个插件里审合同（不过对电脑显卡稍微有点要求）。

### 4. 试着接了点真法条 (MCP)

AI 有个坏毛病就是喜欢胡编法条。如果对此有强迫症，可以外挂一个本地法律数据库（比如开源的 Legal Database）。用了 MCP 协议连上以后，它审合同就有真法条参考了，终于不乱编《民法典》了。

## 🔌 自己带钥匙开门

这个只是个供大家白嫖的开源外壳，本质还是靠各大厂商的 AI 输出。所以没有内置账号，需要你自己带 Key 使用（丰俭由人）：

- **DeepSeek**：不用多说了，便宜大碗，十几万字长上下文，充 10 块钱能跑好久。
- **其他通用的**：像通义千问、Kimi、智谱 GLM 或者国外的 Claude 开发接口，只要填个地址和 Key 都能接进来。

## 🏃‍♂️ 随便玩玩的上手方式

1. 电脑上装个 [Node.js](https://nodejs.org/)（默认下一步就行）。
2. 在我的 GitHub 仓库把代码 ZIP 包下载到桌面，解压。
3. **Windows 用户**双击 `启动.bat`；**Mac 用户**打开终端，输入 `bash` 把 `启动.command` 拖进去回车。
4. 选你要开 Word 还是 WPS，等几秒钟脚本跑完。
5. 去设置里填上你找来的 API Key，收工。
